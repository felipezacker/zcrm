# Design Quality Checklist

## Visual Hierarchy

- [ ] Clear heading hierarchy (H1 > H2 > H3)?
- [ ] Primary CTA visually dominant?
- [ ] Important content above the fold?
- [ ] Eye flow guides to conversion?

## Colors

- [ ] All colors from design tokens (zero hardcoded)?
- [ ] OKLCH color space used?
- [ ] Sufficient contrast (WCAG AA 4.5:1)?
- [ ] Consistent color usage across sections?

## Typography

- [ ] All font sizes from token scale?
- [ ] Max 2 font families (display + body)?
- [ ] Line heights comfortable (1.5-1.7 body)?
- [ ] No orphan/widow lines in key sections?

## Spacing

- [ ] All spacing from token grid (4px base)?
- [ ] Consistent section padding?
- [ ] Breathing room between elements?
- [ ] No cramped or overly spaced areas?

## Consistency

- [ ] Same button style across all CTAs?
- [ ] Consistent card/section patterns?
- [ ] Icons from same library (Lucide)?
- [ ] Image treatment consistent (radius, shadows)?
